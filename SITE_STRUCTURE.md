- テキスト: `詳細な試験データを確認する`

**詳細データ01（還元当量）:**
- タグ: `試験データ 01`
- メイン数値: `23.02 mg/g`
- タイトル: `原料1gあたりの還元当量（中央値）`
- サブ: `参考値：飽和水素水換算 約14.4L/g`
- チャートタイトル: `ハイドロシェル添加による吸光度変化（340nm）`
- バーチャート:
  - NAD+（反応前）: ≈ 0（幅2%）
  - NAD+ + ハイドロシェル（反応後）: 2.006（幅73%）
  - 基準値: NADH標準 0.5 mmol = 2.752
- 測定注記: `測定:山梨県産業技術センター（産技セ第3102号）`
- 補足: `※上記測定データを基に自社にて算出`
- 補足: `※NADH標準液との吸光度比から、反応により生じたNADH量を求め、還元当量（mg/g）に換算しています`

**詳細データ02（水素発生持続時間）:**
- タグ: `試験データ 02`
- メイン数値: `24 h`
- タイトル: `原料試験における測定時間`
- サブ: `30分〜24時間の測定を実施`
- チャートタイトル: `気相水素濃度の推移（GC法）`
- バブルチャートデータ:
  - 30分: 233,000 ppm（バブルサイズ40px、透明度0.25）
  - 1時間: 321,000 ppm（バブルサイズ48px、透明度0.4）
  - 4時間: 510,000 ppm（バブルサイズ56px、透明度0.65）
  - 16時間: 530,000 ppm（バブルサイズ58px、透明度0.8）
  - 24時間: 580,000 ppm（バブルサイズ64px、透明度1.0）
- 試験条件: `試料: 原料1g＋純水5mL / ヘッドスペースバイアル20mL`
- 装置: `島津 GC-2010plus（GC-BID）`
- 測定機関: `一般社団法人 国際水素規格協会（WHSA認証）`

**免責注記:**
`※本データはハイドロシェル原料に関する試験結果です。最終製品そのものの効果効能を示すものではありません。試験条件に基づく測定値です。`

## 6. JavaScript機能

### 6.1 フェードアップアニメーション

- **対象:** `.fade-up` クラスを持つ全要素
- **動作:** IntersectionObserver（threshold: 0.12）で監視、ビューポートに入ったら `.visible` クラスを付与
- **効果:** opacity: 0 → 1、translateY(18px) → 0、transition 0.5s
- **一度のみ:** 表示後は `unobserve` で監視解除

### 6.2 FAQアコーディオン

- **対象:** `.faq-q` ボタン
- **動作:** クリックで `.faq-item` に `.open` を付与/削除、`.faq-a` の maxHeight を制御
- **排他制御:** 1つ開くと他は閉じる

### 6.3 詳細データアコーディオン

- **対象:** `#detailToggle` ボタン
- **動作:** クリックで `#detailContent` の maxHeight を制御
- **SVGアイコン:** 開閉時に180度回転

### 6.4 法的パネルトグル

- **関数:** `togglePanel(id, btn)`
- **対象:** フッターの3つの法的コンテンツ（`#tokushouhou`, `#privacy`, `#terms`）
- **動作:** 排他制御（1つ開くと他は閉じる）

### 6.5 固定CTAバー表示制御

- **対象:** `#fixedCta`
- **動作:** スクロール位置を監視
  - FVの下端より下 かつ 購入セクションの上端より上 → 表示
  - それ以外 → 非表示
- **パフォーマンス:** `{ passive: true }` オプション付き

### 6.6 Stripe Checkout

- **関数:** `handleCheckout(priceType)`
- **動作:**
  1. `/api/checkout` に POST（`{ price_type: 'first' | 'regular' }`）
  2. レスポンスの `url` にリダイレクト（Stripe Checkout画面）
  3. エラー時はalertで通知
- **サーバーサイド（`functions/api/checkout.js`）:**
  - Cloudflare Pages Functions で実行
  - 環境変数: `STRIPE_SECRET_KEY`, `STRIPE_PRICE_FIRST`, `STRIPE_PRICE_REGULAR`
  - Stripe API で Checkout Session を作成
  - 配送先: 日本のみ
  - 決済方法: card（コンビニ・PayPayはコメントアウト）
  - 成功URL: `/?checkout=success`
  - キャンセルURL: `/?checkout=cancel`
